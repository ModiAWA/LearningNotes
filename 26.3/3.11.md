# C - Omelette Restaurant

On day i*i* (1≤i≤N1≤*i*≤*N*) after opening, the following actions were performed.

- In the morning of day i*i*, Ai*A**i* eggs are purchased.

- In the evening of day i*i*, all eggs that have been stocked for D*D* or more days are discarded.

Find how many eggs remain in the restaurant after the evening action on day N*N*.

### Constraints

- 1≤T≤2×1051≤*T*≤2×105

- 1≤Ai,Bi≤101≤*A**i*,*B**i*≤10
- Eggs never run out at noon on any of the N*N* days.
- For each input, the sum of N*N* over all test cases is at most 2×1052×105.
- All input values are integers.

### Input

The input is given from Standard Input in the following format:

```
TT
case1case1
case2case2
⋮⋮
caseTcaseT
```

caseicase*i* represents the i*i*-th test case.
Each test case is given in the following format:

```

A1A1 A2A2 …… ANAN
B1B1 B2B2 …… BNBN
```

### Output

Output T*T* lines.
The i*i*-th line (1≤i≤T1≤*i*≤*T*) should contain the answer for the i*i*-th test case.

### Sample 1

| ------------------------------------------------ | ---------- |
| `3 3 1 7 2 3 1 3 2 3 2 7 2 3 1 3 2 2 1 2 1 1 2 ` | `3 5 0 `   |



In the first test case, the following actions are performed.

- Initially, AtCoder Restaurant has no eggs.
- In the morning of day 11, 77 eggs are purchased. The restaurant has 77 eggs stocked on day 11.
- At noon on day 11, 11 egg is used. The restaurant has 66 eggs stocked on day 11 remaining.
- In the evening of day 11, no eggs are discarded. The restaurant has 66 eggs stocked on day 11 remaining.
- In the morning of day 22, 22 eggs are purchased. The restaurant has 66 eggs stocked on day 11 and 22 eggs stocked on day 22.
- At noon on day 22, 33 eggs are used. The restaurant has 33 eggs stocked on day 11 and 22 eggs stocked on day 22 remaining.
- In the evening of day 22, the eggs stocked on day 11 are discarded. The restaurant has 22 eggs stocked on day 22 remaining.

- At noon on day 33, 22 eggs are used. The restaurant has 33 eggs stocked on day 33 remaining.

Thus, 33 eggs remain after the evening action on day 33, so output 33 on the first line.

For the second test case, remember to output the number of eggs after discarding the eggs stocked on day 11 in the evening of day 33.

两个2*10^5 不能直接模拟（O(n^2)) 用队列模拟O（n) 解决

```c++
#include<bits/stdc++.h>
using namespace std;
#define IOS ios::sync_with_stdio(false); cin.tie(nullptr); cout.tie(nullptr);
#define endl '\n' 
#define ll long long
#define debug(x) cout<< #x << ' ' << x << endl;

const int MAXX = 1e7 + 10;
const int inf = 1e9 + 10;
const ll INF = 1e18 + 10;

void test(){
	int n,d;cin>>n>>d;
	vector<int> buy(n,0),use(n,0);
	for(int i = 0; i < n;i++){
		cin>>buy[i];
	}
	for(int i = 0 ; i<n;i++){
		cin>>use[i];
	}
	queue<pair<int,int>> q;
	ll egg = 0;
	for(int i = 0 ; i<n;i++){
		if(buy[i]>0){
			q.push({i,buy[i]});
			egg += buy[i];
		}
		int used = use[i];
		while (used > 0 && !q.empty()){
			auto &p = q.front();
			if(p.second <= used){
				used -=p.second;
				egg -= p.second;
				q.pop();
			}
			else {
				p.second -=used;
				egg -=used;
				used = 0;
			}
		}
		while(!q.empty()&&i - q.front().first >= d){
			egg -= q.front().second;
			q.pop();
		}
	}
	cout<<egg << endl;
	

}

int main(){
	IOS
	int t ;cin >> t;
	while(t--){
		test();
	}
	 return 0;
}
```

		next[0][i]=a[i];
	}
	for(int k = 1; k < step;k++){
		for(int i = 0; i < n;i++){
			next[k][i] = next[k-1][next[k-1][i]];

	}

	for(int i = 0 ; i < n;i++){
		int pos = i;
		ll steps = 1e18;
		for(int k = 0; k < step;k++){
			if(steps >> k&1){
				pos = next[k][pos];
			}
		}
		ans[i] = pos+1;
	}
	for(int i = 0 ; i < n;i++){
		cout<<ans[i]<<' ';
	}
}
